# Single Sign On (SSO) Example with Spring Boot - Authentication Service
## What you'll need
- JDK 1.7+
- Maven 3+

## Stack
- Java
- Spring Boot
- FreeMarker

## Run
- Run Authentication Service: webserver (port:8080)
- Run Resource Service 1: web1 (port: 8180)
- Run Resource Service 2: web2 (port: 8280)
